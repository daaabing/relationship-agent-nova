# V1 Scope

Must include:

1. Upload recent Apple Notes (14 days default)
2. Extract people and confirm aliases
3. Generate Open Loop Inbox (5-10 items)
4. Each loop contains:
   - Evidence snippet
   - Loop type
   - Suggested next action
5. Done/Dismiss updates persistent state
6. Meeting Prep Mode:
   Input: person name
   Output:
     - last mention summary
     - unresolved threads
     - conversation suggestions

Non-goals:
- Screenshot ingestion
- Voice mode
- Calendar integration
- External data scraping
