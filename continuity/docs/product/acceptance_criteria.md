# Acceptance Criteria

Open Loop Inbox:
- Displays 5-10 items max (configurable)
- Sorted by priority score
- Each item includes:
  - Evidence text
  - Loop category
  - Suggested action

Loop categories must include:
- Promise
- Follow-up
- Decision Pending
- Tension
- Recurring Pattern
- Prep Needed

State:
- Done loops removed from active list
- Dismissed loops removed from active list
- State persists after refresh

Meeting Prep:
- Shows last interaction summary
- Shows unresolved loops
- Shows suggested conversation starters

Traceability:
- Every generated insight must reference original note text.
